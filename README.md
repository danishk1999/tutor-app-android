# 📱 Tutor App — Android
**Course:** CMPT 395 — Software Engineering Project  
**Institution:** MacEwan University  
**Status:** ✅ Completed

## Overview
A full-featured Android tutoring platform that connects students with tutors. 
Built using Kotlin with a Firebase backend for real-time data storage and 
user authentication. Developed as a team software engineering project following 
an agile development process.

## Features
- User registration and authentication via Firebase Auth
- Student and tutor profile creation and management
- Browse and search available tutors by subject
- Booking and scheduling tutoring sessions
- Real-time database updates using Firebase Firestore
- Intuitive Android UI following Material Design guidelines

## Technologies Used
- **Kotlin** — Primary programming language
- **Android Studio** — Development environment
- **Firebase Authentication** — User login and registration
- **Firebase Firestore** — Real-time cloud database
- **Firebase Storage** — Profile image storage
- **Material Design** — UI/UX design guidelines

## Architecture

Android Frontend (Kotlin)
↓
Firebase Backend
├── Authentication
├── Firestore Database
└── Storage
